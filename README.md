# Stokyr — Smart Inventory Management 📦

O **Stokyr** é um sistema de gerenciamento de estoque focado em simplicidade, eficiência e usabilidade.

O projeto foi criado para substituir controles manuais e planilhas complexas, oferecendo uma experiência moderna para gestão de produtos, movimentações de estoque e monitoramento de inventário em tempo real.

A aplicação será desenvolvida utilizando uma arquitetura separada entre backend e frontend, com foco em organização, escalabilidade e experiência do usuário.

---

# 🚀 Funcionalidades

## ✅ MVP (Versão Inicial)

- Cadastro de produtos
- Controle de entrada e saída de estoque
- Histórico de movimentações
- Cadastro de fornecedores
- Dashboard com métricas principais
- Alertas de estoque baixo
- Upload de imagem para produtos
- Sistema de autenticação

# 🧱 Arquitetura

O projeto será dividido em duas aplicações independentes:

```txt
stokyr-backend/
stokyr-frontend/
```

### Backend
Responsável por:
- regras de negócio
- autenticação
- gerenciamento de estoque
- persistência de dados
- API REST

### Frontend
Responsável por:
- interface do usuário
- dashboard
- visualização de métricas
- experiência responsiva
